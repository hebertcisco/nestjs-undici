## Installation

> Install with yarn or npm: `yarn` or `npm`:

```bash
# yarn
yarn add nestjs-undici
```

```bash
# npm
npm i nestjs-undici --save
```

### Import the lib with es6 or cjs:

```mjs
// es6
import { HttpModule } from 'nestjs-undici';
```
